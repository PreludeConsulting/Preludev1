# Career Data Schema (O*NET + BLS OOH)

## O*NET Web Services
- URL: <https://services.onetcenter.org/>
- Use for:
  - Occupation discovery
  - Skills
  - Tasks
  - Interests
  - Work activities
  - Tools and technologies

## BLS Occupational Outlook Handbook
- URL: <https://www.bls.gov/ooh/>
- Use for:
  - Median pay
  - Job outlook (% growth)
  - Typical entry-level education
  - Duties
  - Related occupations

## Example verified BLS record
- Occupation: Software Developers, Quality Assurance Analysts, and Testers
- Source page: <https://www.bls.gov/ooh/computer-and-information-technology/software-developers.htm>
- Metrics (reference snapshot):
  - 2024 median pay: $131,450/year
  - Typical entry-level education: Bachelor's degree
  - Outlook (2024-2034): 15% (much faster than average)

## Usage rules
1. Use O*NET for "what this career does" and fit exploration.
2. Use BLS for salary/outlook/education claims.
3. If career metrics are not retrieved, say verified metrics are unavailable in current context.
