# Optional IPEDS add-on

Prelude should begin with College Scorecard and O*NET because those sources cover the first practical features:

- college search
- cost comparison
- basic admissions context
- major and career exploration

IPEDS is valuable when you need more detailed institutional variables.

Official pages:

- https://nces.ed.gov/ipeds/use-the-data
- https://nces.ed.gov/ipeds/help/complete-data-files

IPEDS complete data files are selected by **survey** and **year** and are downloaded as ZIP archives containing CSV files. Add only the files required by a specific feature rather than importing every historical survey file at once.

Potential future uses:

- more detailed enrollment breakdowns
- staffing and institutional characteristics
- detailed financial-aid statistics
- degrees awarded by program
